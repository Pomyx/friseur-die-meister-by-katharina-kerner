# Friseur Die Meister by Katharina Kerner

Premium business website created for a real hair salon in Balingen, Germany.

A custom-built modern web presence focused on elegance, trust, speed and mobile usability.

---

## Live Website

https://pomyx.github.io/friseur-die-meister-by-katharina-kerner/

---

## Project Vision

This website was designed to give a local salon a premium digital identity with the atmosphere of a luxury brand.

The goal was to create a site that feels:

- Elegant  
- Modern  
- Warm  
- Trustworthy  
- High-end  
- Easy to use on mobile devices

---

## Core Features

- Fully responsive mobile-first layout
- Premium dark / gold branding
- Cinematic hero sections
- Fast loading static architecture
- Custom navigation system
- Local hosted fonts (privacy optimized)
- GDPR-friendly 2-click Google Maps integration
- Legal pages (Impressum / Datenschutz)
- SEO-ready semantic structure
- Smooth scrolling experience

---

## Tech Stack

- HTML5  
- CSS3  
- Vanilla JavaScript  
- GitHub Pages Hosting

---

## Design Direction

Luxury salon aesthetic with clean typography and refined contrast.

Visual language includes:

- Black premium background
- Gold accent colors
- Elegant serif headings
- Modern sans-serif readability
- Spacious layout rhythm
- Strong mobile presentation

## Performance & Privacy

Special attention was given to modern lightweight performance and privacy:

- No bloated frameworks
- No unnecessary trackers
- Locally hosted fonts
- Deferred Google Maps loading
- Clean static deployment
